# sample-fizzbuzz-test

`Node.js` and `npm` are required, so please install them beforehand.

## Usage

```shell

# clone this repo
$ git clone git@github.com:kkeeth/sample-fizzbuzz-test.git

# install packages
$ npm install

or

$ yarn

# execute tests
$ npm run test

or

$ yarn test

# execute fizzbuzz
$ node fizzbuzz.js
```
